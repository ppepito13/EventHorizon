# **App Name**: EventHorizon

## Core Features:

- Event Registration: Allow users to register for the currently active event via a dynamic form tailored to the event's specific data requirements.
- Dynamic Form Generation: Generate event-specific registration forms based on the data requirements defined in the admin panel. Form fields and validation rules should be configurable.
- Event Management Dashboard: Provide an admin interface for authorized users to create, modify, and delete event pages. Include options to toggle event visibility (enable/disable access).
- Themed Event Pages: Each event page should have a unique visual theme based on the specifications defined by the administrator, including custom graphics and RODO (GDPR) information.
- AI Content Generation: Use AI to help generate engaging event descriptions and promotional copy based on event details entered by the administrator. The AI acts as a writing tool, suggesting text variations to the admin.

## Style Guidelines:

- Primary color: Dark Teal (#002E3C) for a sophisticated and modern look, mirroring the provided screenshot.
- Accent color: Gold (#FFD700) to highlight key elements and calls to action, as per the provided screenshot.
- Background color: Very light desaturated teal (#E0F4F7), offering a subtle contrast with the dark primary color.
- Headline font: 'Space Grotesk', a sans-serif, for a tech-forward and clear title and heading aesthetic.
- Body font: 'Inter', a sans-serif, for readability and a modern user experience.
- Use minimalist icons to represent event categories or specific event features, ensuring they align with the overall modern aesthetic.
- Emphasize a single active event with a prominent display. The registration form is prioritized for direct user engagement. Utilize a clean and structured layout with sufficient spacing to prevent visual clutter.
